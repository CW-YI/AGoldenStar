<h1> AGoldenStar </h1>
세종대학교 게임 제작 동아리 판도라큐브에서 진행한 승급 프로젝트입니다.<br>
2023년 하반기에 90일동안 개발한 1인 프로젝트로, Unity 2D를 사용하여 개발하였습니다.<br>
개발 후, 깃허브에 업로드했습니다.<br>
<br>

<h2> 게임 개요 </h2>
액션 어드벤쳐 게임으로, 각 던전들을 클리어하며 별 조각들을 모두 모아 엔딩 지점에 도달하는 것이 목표입니다.<br>
